MyGym es una app que busca lograr una integridad entre el usuario y la administración de un local/Gym/profesor particular.

Esta integridad la logra creando un sector para que el usuario busque y encuentre el gym que se adapte a sus necesidades sumándole un seguimiento por parte del local a este usuario que esta inscripto a dicho local, dentro de este seguimiento hay cosas como lista personalizada de los ejercicios que debe realizar los días y hora en las que le toca entrenar, fecha de pagos, etc.
